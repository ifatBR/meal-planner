# Design Reference

Exported frames from the Flower Figma UI kit used as the design system base for this project.
The token values derived from these frames are defined in `src/styles/tokens.css`.

## Files

- `color.png` — color palette (primary, supporting, secondary colors)
- `font.png` — typography scale and text styles
- `buttons_inputs_all_components.png` — button variants, input fields, all components
- `avatar_dropdown_etc.png` — avatars, dropdowns, tooltips, labels, misc components
- `icon_and_emojy.png` — icon set and emoji reference
- `sidebar.png` — sidebar variants (light mode used in this app)
- `components.png` — component overview and list/card patterns
- `calendar_week.png` — calendar week view reference layout
- `contacts_list.png` — list view inspiration with detail panel
- `project_management_list.png` — table/list view with status, progress, pagination
- `task_details.png` — modal/popup pattern with background overlay
- `todo_list.png` — dark mode list item pattern
